How it works:
1. Enter your Jira URL and project keys above.
2. Click ① Open Jira Login — this opens Jira in a new tab so your browser gets a session cookie.
3. Log in to Jira in that tab, then come back here.
4. Click ② Fetch & Generate — calls the Xray REST API on your Jira server using your browser session (no proxy issue, no API keys needed).

⚠ This page must be opened from the same origin as your Jira server, or served locally alongside it. If you see CORS errors, use the browser extension or host this file on your Jira server.
